# Конфигурационные файлы
1) `proxy.conf` - файл конфигурации nginx для проксирования запросов. /api/* проксирует на порт 8080 бэкенда,
/* проксирует на 5173 фронтенда.
2) `frontend.conf` - файл конфигурации nginx для фронтенда.
3) `backend.env` - переменные окружения для бэкенда. Полная структура будет описана позже. Пока( думаю названия очевидные):
```
DRONE_CORS_ORIGINS=*
```
4) `elastic.env` - переменные окружения для Elasticsearch.
5) `init-elastic.yaml` - конфигурационный файл для init-контейнера ElasticSearch. Поддерживает одно поле:
`elastic_url: http://elastic:9200`

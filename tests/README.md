# Описание
Хранятся тесты для кода, в отдельных папках для отдельных частей кода. По умолчанию включено показание покрытия тестов по файлам.

# Запуск(временно)
## 1. Убедитесь, что контейнеры запущены
docker compose up -d backend elastic

## 2. Установите зависимости для тестов
pip install -r tests/requirements.txt

## 3. Запустите тесты
pytest tests/endpoints/ -v -s

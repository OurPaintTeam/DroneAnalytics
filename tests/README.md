# Описание
Хранятся тесты для кода, в отдельных папках для отдельных частей кода. По умолчанию включено показание покрытия тестов по файлам.
# Запуск
1. Зайти в папку tests
2. Запустить uv sync, для выкачки зависимостей
3. Потом запустить тесты. Вот команда для запуска всех тестов: uv run pytest. Потом запускать через неё.

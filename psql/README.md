# Описание
## Модель
все что нужно для рядового зеваки можно увидеть в диаграмме - ER.pdf и в дампе - dump.sql
## Запуск
1) dump.sql - очевидно
2) secret.sql - все что будет помещено в github секрет. Как минимум пароль и роли к пользователю
3) .env - админка
4) Надо использовать volume data:/var/lib/postgresql/data
5) Порт дефолтный - 5432
## Использование
Для теста использовать свои имена и пароли. Обязательно пробрасывать все через файлы окружений. Они после будут заменены
на нужные через секреты. Также приложения не будут иметь возможность ничего создавать и удалять(DROP) - настраивайте 
библиотеки соответственно.

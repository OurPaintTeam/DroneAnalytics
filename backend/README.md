# Описание
Слово backend говорит само за себя. Отвечает за системный журнал и инфопанель -
информацию можно подчерпнуть из [api](/docs/api.yaml) спецификации. Также на него лежит участь в хранении паролей и апи ключей. На данный момент, для версии "на заглушках" апи ключ пробрасывается через
переменную окружения.
## Запуск
 - порт 8080
 - .env - переменные окружения
